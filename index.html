Facebook Clone

Project Description: This is a React clone of the Facebook website. I implemented a live database for the feed and user authentication using Firebase. The icons that appear in this project were obtained using Material-UI. The app state was managed with Context API. The website renders once the user logs in with their google account. The user can also post images by pasting a url in the input field. Users can communicate with each other through the live feed.

Technologies Used: React, Context API, Hooks, Firebase, Material-UI and CSS.

Netflix Clone

Project Description: This project is a React clone of the Netflix website. The UI is styled to closely resemble the Netflix UI.
